# 🏏 IPL Data Analytics Project

## 📌 Project Overview

This project presents an end-to-end analysis of the Indian Premier League (IPL) using Python, SQL, and Tableau. The objective was to clean, analyze, and visualize IPL match and player data to uncover meaningful insights into team performance, player achievements, and match outcomes through interactive dashboards.

---

## 🎯 Business Objectives

- Analyze overall IPL team performance.
- Identify the highest run scorers and wicket takers.
- Find players with the most fours and sixes.
- Study the impact of toss decisions on match results.
- Explore season-wise match trends.
- Build interactive dashboards for cricket analytics.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- SQL (MySQL)
- Tableau Public
- Jupyter Notebook

---

## 📂 Dataset

The project uses two IPL datasets:

- **matches.csv** – Match-level information
- **deliveries.csv** – Ball-by-ball match data

Dataset Source:
https://www.kaggle.com/datasets/nowke9/ipldata

## 🔍 Key Insights

- Mumbai Indians emerged as the most successful team based on total match wins.
- Virat Kohli scored the highest number of runs.
- Lasith Malinga finished as the leading wicket-taker.
- Chris Gayle hit the highest number of sixes.
- Toss decisions influenced match outcomes in a significant number of games.
